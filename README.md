# Smart Intrusion Detection System

This project implements a Smart Intrusion Detection System using an ESP32 microcontroller, PIR motion sensor, LED indicator, buzzer, and ThingSpeak cloud platform.

The system continuously monitors a restricted area for human movement. When motion is detected, the ESP32 activates local alerts through an LED and buzzer while simultaneously sending real-time data to the ThingSpeak cloud platform over Wi-Fi. The cloud dashboard enables remote monitoring and visualization of intrusion events.

The project demonstrates the practical application of Internet of Things (IoT) technology in security and surveillance systems. It provides a low-cost, scalable, and efficient solution for detecting unauthorized access and generating instant alerts.

# Features

* Real-time motion detection using PIR sensor
* ESP32-based processing and control
* LED visual alert indication
* Buzzer audible alert system
* Wi-Fi connectivity
* ThingSpeak cloud integration
* Real-time monitoring dashboard
* Wokwi simulation support

# Technologies Used

* ESP32 DevKit V1
* PIR Motion Sensor
* Arduino IDE
* Wokwi Simulator
* ThingSpeak Cloud Platform
* HTTP Communication Protocol
